# AirMouse

#### Pinout:
| NodeMCU pin   | BNO055 |
| ------------- | ------------- |
| D1  | SCL (BNO) |
| D2  | SDA (BNO) |
| 3V3 | VCC (BNO) |
| GND | GND (BNO) |

| NodeMCU pin   | Li-ion battery |
| ------------- | ------------- |
| VIN | BAT+ |
| GND | BAT- |

| NodeMCU pin   | Button |
| ------------- | ------------- |
| GND | pin 1 |
| D4 | pin 2 |
